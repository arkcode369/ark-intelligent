## New Findings

### 1. context.Background() Without Timeout in impact_recorder.go (Medium Priority)

**File:** `internal/service/news/impact_recorder.go:108`

```go
if afterTime.After(time.Now()) {
    go r.delayedRecord(context.Background(), ev, mapping.ContractCode, beforePrice, surpriseSigma, sigmaBucket, horizon, duration)
    continue
}
```

**Issue:** Similar to TASK-CODEQUALITY-003 in chat_service.go, this goroutine uses `context.Background()` without timeout for delayed impact recording. If the API hangs, this goroutine will leak.

**Risk:** Goroutine leak if price API hangs during delayed recording  
**Fix:** Wrap in `context.WithTimeout(context.Background(), 5*time.Minute)`
